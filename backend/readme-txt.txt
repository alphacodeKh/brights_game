Electronic journal plugin for educational institutions with Advanced Custom Fields Pro support and Excel import functionality.

== Description ==
Plugin for managing student grades and educational data with Excel import support.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/electronic-journal` directory
2. Install required dependencies using Composer:
   cd /wp-content/plugins/electronic-journal
   composer install
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Make sure Advanced Custom Fields Pro is installed and activated

== Frequently Asked Questions ==
= What file formats are supported for import? =
The plugin supports .xlsx and .xls files.
